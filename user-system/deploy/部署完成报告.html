🚀 3步完成部署

第一步：初始化数据库

在宝塔面板中执行以下操作：

1. 登录宝塔面板
2. 进入"数据库" -> "MySQL"
3. 点击"添加数据库"
4. 填写信息：
- 数据库名: zizai_user_system
- 用户名: zizai_admin
- 密码: 设置安全密码
- 字符集: utf8mb4
5. 点击"提交"
6. 导入数据库脚本：
- 选择刚创建的数据库
- 点击"导入"
- 上传文件: database-init.sql
- 点击"执行"

第二步：配置API连接

修改配置文件 api/config.php：

// 数据库配置
define('DB_HOST', '127.0.0.1');
define('DB_PORT', '3306');
define('DB_NAME', 'zizai_user_system');
define('DB_USER', 'zizai_admin');
define('DB_PASS', '你的数据库密码');
define('DB_CHARSET', 'utf8mb4');

// JWT密钥
define('JWT_SECRET', '生成随机字符串');
define('JWT_EXPIRY', 86400 * 7);

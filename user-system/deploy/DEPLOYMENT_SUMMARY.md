## 🔐 安全特性

- **资源权限**: 文件所有者、公开/私密控制

### 数据安全
- **密码加密**: BCrypt强哈希加密存储
- **SQL防护**: 参数化查询防止注入
- **XSS防护**: 输入输出过滤
- **CSRF防护**: Token验证
- **文件验证**: 类型、大小、MIME类型验证

### 存储安全
- **文件隔离**: 用户文件独立目录存储
- **权限控制**: 文件访问权限验证
- **存储限制**: 10GB个人存储空间

## 🚀 核心功能模块

### 1. 用户管理系统
- **用户注册**: 用户名、邮箱、密码、个人简介
- **用户登录**: JWT Token认证，记住我功能
- **个人资料**: 头像上传、昵称修改、个人简介
- **密码管理**: 修改密码、密码强度验证
- **注册审核**: 管理员审核新用户注册申请

### 2. 管理员后台
- **用户管理**: 查看所有用户、批准/拒绝用户、删除用户
- **注册审核**: 审核新用户注册申请，查看申请理由
- **存储管理**: 为用户分配存储空间，查看使用情况
- **系统统计**: 用户统计、文件统计、活跃度分析
- **操作日志**: 查看所有用户的操作记录

### 3. 作品画廊系统
- **作品展示**: 图片上传、缩略图生成、作品详情
- **分类标签**: 作品分类管理、标签系统
- **互动功能**: 点赞、评论、浏览统计
- **权限控制**: 公开/私密作品管理
- **搜索功能**: 标题、描述全文搜索

### 4. 云盘存储系统
- **文件上传**: 支持2GB以下文件上传
- **文件管理**: 文件列表、搜索、排序、筛选
- **文件分享**: 公开/私密文件分享控制
- **下载统计**: 文件下载次数统计
- **存储监控**: 实时存储空间使用情况

## 🔧 配置文件说明

### `api/config.php` 关键配置
```php
// 数据库配置
define('DB_HOST', '127.0.0.1');
define('DB_NAME', 'zizai_user_system');
define('DB_USER', 'zizai_admin');
define('DB_PASS', '你的密码');

// JWT配置
define('JWT_SECRET', '生成随机字符串');
define('JWT_EXPIRATION', 86400); // 24小时

// 文件上传配置
define('MAX_FILE_SIZE', 2 * 1024 * 1024 * 1024); // 2GB
define('UPLOAD_DIR', 'wwwroot/uploads/');
```

### 宝塔面板配置
1. **PHP设置**:
   - upload_max_filesize = 2048M
   - post_max_size = 2048M
   - max_execution_time = 300

2. **Nginx配置**:
   - client_max_body_size 2048M;
   - 伪静态规则配置

## 🚨 故障排除

### 常见问题及解决方案

**问题1: 数据库连接失败**
```
原因: 数据库配置错误
解决: 检查 api/config.php 中的数据库连接信息
```

**问题2: 文件上传失败**
```
原因: 上传目录权限不足或PHP配置限制
解决: 检查上传目录权限，修改PHP的upload_max_filesize
```

**问题3: 404错误**
```
原因: 伪静态配置错误
解决: 检查Nginx配置，确保URL重写正确
```

**问题4: 权限不足**
```
原因: 用户权限不足
解决: 使用管理员账户 xiaoxin/zizai123 登录
```

**问题5: JWT认证失败**
```
原因: JWT密钥不匹配或Token过期
解决: 检查JWT配置，重新登录获取新Token
```
